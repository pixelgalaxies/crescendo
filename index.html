Crescendo 2

A flexbox grid framework

Mobile First

Crescendo was built with a mobile first approach. This allows for fully-responsive functionaility when building your websites.

Lightweight

Crescendo is extremely lightweight, perfect for keeping your site running at lightning speeds. The regular CSS file is _kb and the minified file is only _kb.

Open Source

Not only is Crescendo built with a mobile first philosophy and incredibly lightweight, it is also 100% open source.

Support

Crescendo 2 is proud to announce it will be supported by all modern browsers. This includes Chrome, Opera, Firefox, Edge, and finally, Safari.

Is Crescendo right for you?

The simple answer is yes, it is. Crescendo features simple to use grids which are perfect for any project, big or small. It's two grids combined with it's several utility classes are just what you need to build fast, responsive, yet elegant websites. Visit the API page here and learn to use Crescendo today.

New Utility Classes

A couple of new utility classes were added to the roster.

Introducing Round

This is the .round class. It curves its corners at 50px.

Introducing Circle

This is the .circle class. It makes its item, you guessed it, circular.

Buttons

In Crescendo 2, we did away with ghost buttons and added some buttons everyone should be familiar with. We also reintroduce the .btn class with a couple changes. First we set a fixed width of 120px. Then we gave it a set padding of 10px. Lastly we set the font-size to 15px. After the changes to the .btn were made, we added the secondary classes you see below.
