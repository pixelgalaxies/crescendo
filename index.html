Crescendo

A flexbox grid framework

Mobile First

Crescendo was built with a mobile first approach. This allows for fully-responsive functionaility when building your websites.

Lightweight

Crescendo is extremely lightweight, perfect for keeping your site running at lightning speeds. The regular CSS file is 6kb and the minified file is only 4kb.

Open Source

Not only is Crescendo built with a mobile first philosophy and incredibly lightweight, it is also 100% open source. View it here.

Support

Crescendo is proud to announce it will be supported by all modern browsers. This includes Chrome, Opera, Firefox, Edge, and finally, Safari.

Why use Crescendo?

Crescendo features simple to use grids which are perfect for any project, big or small. Its two grids combined with its several utility classes are just what you need to build fast, responsive, yet elegant websites. Visit the API page here and start using Crescendo today.
