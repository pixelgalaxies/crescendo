Crescendo

A flexbox grid framework.

Mobile First

Crescendo was with a mobile first approach in mind. It allows for fully-responsive functionaility when building your websites.

Lightweight

Crescendo is extremely lightweight, perfect for keeping your site running at lightning speeds. The regular CSS file is _kb and the minified file is only _kb.

Open Source

Not only is Crescendo built with a mobile first philosophy and incredibly lightweight, it is also 100% open source. You can find it here.

Support

Crescendo is supported by all modern browsers except Safari (for now).

Is Crescendo right for you?

The simple answer is yes, it is. Crescendo features simple to use grids which are perfect for any project, big or small. It's two grids combined with it's several utility classes are just what you need to build fast, responsive, yet elegant websites.

Buttons

There are two button styles that each have unique propeties. The .btn class and it's sister classes, .ghost-btn and .default-btn, can be applied to a button tag, an input['submit'] tag, an input['button'] tag, and lastly an anchor tag. More button classes will be added in future versions of Crescendo.

Ghost Button

The first button class is the Ghost Button. It features a transparent background with a black border and matching text. When you roll over this button it will fill in black and the text will turn white. You can of course override the colors in a separate stylesheet.

Anchor

Default Button

The second button class is the Default Button. It features a dark gray background with off-white text. Like the ghost button, you're able to override the colors in a separate stylesheet.

Utility Classes

These are just a couple classes you may use in your grids, rows, and columns.

.inner-between

Use this class in a new div placed inside a column.

.inner-around

Use this class in a new div placed inside a column.

This is the .end class.

This class goes alongside your .col* class. It will send all of your content to the bottom of the column.

This is the .start class.

This class goes alongside your .col* class. It will set all of your content to the top of the column.

This is the .just-center class.

This class goes alongside your .col* class. It will send all of your content to the middle of the column.

This is the .right class.

This class goes alongside your .col* class. It will align all of your content to the right of the column.

This is the .left class.

This class goes alongside your .col* class. It will align all of your content to the left of the column.

This is the .center class

This class goes alongside your .col* class. It will align all of your content to the center of the column.
